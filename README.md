## 🔬 Undergrad Research & Featured Projects

### 1. 📊 Behavior-Aware Personal Budget Tracker & Research Artifact
*An active undergraduate research project exploring the "Explainability Gap" in FinTech under the advisement of Prof. Vladislav D. Veksler.*
*   **The Core System:** Built a full-stack personal finance application with a **Streamlit** dashboard front-end and a secure **FastAPI** backend storing transaction data in an optimized **SQLite** layout.
*   **The Research Engine:** Engineered a deterministic Python **Rule Engine** to analyze synthetic consumer profiles generated via **Pandas**. It identifies complex financial vectors like *Subscription Creep* and *Temporal Spikes*.
*   **Advanced Analytics:** Implemented an **Isolation Forest** anomaly detection algorithm paired with regression threshold logic to isolate the top 1.5% of erratic transactions, translating mathematical anomalies into human-readable, prescriptive behavioral coaching flags.
*   *Tech Stack: Python, FastAPI, Streamlit, Pandas, Scikit-Learn (Isolation Forest), SQLite*

### 2. 🛡️ AI Compliance Firewall (LLM Guardrails Middleware)
*A high-throughput security and compliance proxy layer designed to screen interactions between users and open-source LLMs.*
*   **Multi-Tier Scanning:** intercepting user prompts and LLM generations utilizing a native regex scanner running alongside an **Ollama (nomic-embed-text)** embedding layer to catch semantic paraphrasing via cosine similarity.
*   **Knowledge Graph Policy Engine:** Interfaced with a **Neo4j** graph database to pull context-specific compliance disclaimers (e.g., HIPAA/FINRA frameworks) and dynamically inject them into or redact them from **Ollama (Llama 3.1 8B)** outputs.
*   **High-Performance Acceleration:** Features an optional high-performance regex rule engine compiled natively in **Rust** using **PyO3** and **Maturin** to minimize latency overhead.
*   *Tech Stack: FastAPI, FastHTML, HTMX, Ollama, Neo4j (Docker), Rust, PyO3, Pytest, SQLite*
